# Tailwind Shopping Cart

A **responsive shopping cart** built with **Tailwind CSS**.

## Technology Stack

- **Frontend**: HTML, **Tailwind CSS**
- **Languages Supported**: Persian (Farsi) for RTL layout
- **Styling**: Tailwind CSS for fast, responsive design
